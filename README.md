# Job Search Management System

A structured approach to finding and applying for positions while maintaining your authentic voice.

## Directory Structure

```
├── profiles/              # Job profiles you're targeting
├── base-documents/        # Master resume and cover letter
├── applications/          # Individual applications (one folder per application)
├── templates/             # Reusable content modules
├── scripts/               # Automation helpers
├── application-tracker.md # Central tracking document
└── workflow.md            # Document customization process
```

## Quick Start

1. **Define your job profiles** in [profiles/](profiles/)
   - Create one file per profile type you're targeting
   - Use [profiles/TEMPLATE.md](profiles/TEMPLATE.md) as a starting point

2. **Add your base documents** to [base-documents/](base-documents/)
   - These are your master resume and cover letter
   - Keep them authentic to your voice and experience

3. **Create reusable content modules** in [templates/](templates/)
   - Key achievements, skills, project descriptions
   - Makes customization faster while maintaining consistency

4. **Follow the workflow** in [workflow.md](workflow.md) when applying
   - Ensures consistent, voice-authentic customization
   - Prevents overclaiming or misrepresentation

5. **Track everything** in [application-tracker.md](application-tracker.md)
   - One source of truth for all applications
   - Helps with follow-ups and interview prep

## Principles

- **Authentic voice**: All materials sound like you
- **Honest representation**: No unsupported claims
- **Efficient customization**: Reuse proven content
- **Organized tracking**: Know what you sent where
