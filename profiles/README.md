# Job Profiles

Define the different types of roles you're targeting. Each profile helps you:
- Quickly assess if a job posting matches your goals
- Customize documents efficiently
- Track which types of roles are getting traction

## Creating a New Profile

1. Copy [TEMPLATE.md](TEMPLATE.md) to a new file
2. Name it descriptively (e.g., `senior-product-manager.md`, `technical-lead.md`)
3. Fill in each section authentically
4. Update as you learn what resonates

## How to Use Profiles

**When you find a job posting:**
1. Read it thoroughly
2. Ask: "Which profile(s) does this match?"
3. Use that profile's guidance to customize your application
4. Track results to learn which profiles are working

## Example Profile Structure

You might have profiles like:
- `senior-product-manager-b2b.md` - For B2B SaaS PM roles
- `technical-program-manager.md` - For TPM roles at larger companies
- `founding-product-manager.md` - For early-stage startup PM roles

Each profile captures what makes you strong for that type of role and how to present yourself.

## Tips

- **Be specific**: The more focused each profile, the more useful it is
- **Be honest**: Only include experience and skills you actually have
- **Update regularly**: Adjust as you learn what's working
- **2-5 profiles**: More than 5 might mean you're too scattered
