---
name: Role Ordering — Always Reverse Chronological, No Gaps
description: All roles included in a resume must appear in strict reverse chronological order with no timeline gaps between included roles
type: feedback
---

Whatever roles are displayed must appear in reverse chronological order — most recent first. Skipping roles is acceptable; displaying them out of order is not.

**Why:** Placing a recent role (e.g., Solace 2022–2023) in an "Earlier Career" section after an older role (Jaspersoft 2012–2020) makes the timeline look wrong and buries recent experience.

**How to apply:**
- Any role included in the resume must appear before all roles with earlier end dates
- Omitting roles entirely is fine — "Relevant Experience" header covers selective inclusion
- The Earlier Career section is for oldest roles only; never place a role there if a more recent role is listed above it in the main section
